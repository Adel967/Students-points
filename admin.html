إضافة طالب جديد

بعد الإضافة، اذهب لتبويب "تعديل النقاط" لإدخال نقاط الطالب الجديد، ثم "تصدير البيانات" لنسخ الكود.

📖 كيف تحدّث النقاط أسبوعياً؟

  1. افتح صفحة admin.html من المشروع
  2. في تبويب تعديل النقاط، عدّل بيانات كل طالب
  3. اضغط "حفظ وتصدير"
  4. انتقل لتبويب "تصدير البيانات" وانسخ الكود
  5. افتح ملف students.js في مستودع GitHub
  6. اضغط ✏️ تعديل، والصق الكود الجديد، ثم Commit changes
  7. الموقع يتحدث تلقائياً خلال دقيقة! 🎉

🌐 كيف ترفع الموقع على GitHub Pages؟

  1. أنشئ حساباً على github.com
  2. أنشئ مستودعاً جديداً (Repository)
  3. ارفع الملفات الثلاثة: index.html وadmin.html وstudents.js
  4. اذهب لـ Settings → Pages → Source → Main Branch
  5. رابط موقعك: https://اسمك.github.io/اسم-المشروع
